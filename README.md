# 🎬 Movies Library - Verzel

Aplicação de biblioteca de filmes com autenticação e favoritos.

## 🚀 Tech Stack
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Auth:** BetterAuth
- **API:** TMDb API v3

## 📦 Em Desenvolvimento
Este projeto está sendo desenvolvido como teste técnico para a Verzel.
